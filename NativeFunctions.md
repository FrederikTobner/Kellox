# Native functions
### clear
Clears the console e.g.
```
clear();
```
### clock
Get the current time (right now only seconds) e.g.
```
var time = clock();
```
### read
Reads input from the user until he presses enter e.g.
```
var input = read();
```
### wait
Suspends the program for x seconds e.g. 
```
wait(5.25);
```
